以下、使い方

1:変換したいテキストファイルのところへ「LINKS_Tool.exe」をコピー。その後、クリックして実行。

2:「ファイル名を指定してください　：」と表示されるので編集したいファイル名を入力してください。

3:ファイルを読み込んで、画面に出力します。
　「B01」「C02」「V05」「M07」「S09」などの演出タグとコメントタグ「//」などは赤字で出力されます。

4:あとは編集したい行番号を指定し入力していきます。

5:saveと入力すると変更を保存。insertと入力することで、指定の行に文章を挿入できます。
　exitで終了します(saveで保存していない場合、変更は保存されません！)。
